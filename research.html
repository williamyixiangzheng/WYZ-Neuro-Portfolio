Research

Current research directions across hippocampal coding, social territory, and neural devices.

Main Projects

Project Drift

A hippocampal population code for rapid generalization

Pre-print

CA1 population activity uses a disentangled code for location, time, and choice, while CA3, MEC, and local PV-interneurons shape remapping and representational drift.

Project page Preprint

• Mouse behavioral tasks, Arduino coding, optogenetic fiber surgery, perfusion, and imaging.

• Topological data analysis, dimensionality reduction, and hippocampal neural manifold analysis across familiar vs. unfamiliar tracks and long-term (~300-day) recordings with flexible probes.

Project Territory

A socio-spatial cognitive map of a naturalistic environment

In prep / unpublished manuscript

Male mice in a semi-natural outdoor enclosure, studying how hippocampal place-cell activity evolves during territory formation, defense, and intrusion.

Project page

• Neuropixels silicon-probe surgeries and naturalistic outdoor behavioral tasks using wireless neuro-datalogger setups.

• 3D-printed headstages and automated pipelines for multi-animal identity tracking (trajectory + IMU, PCB/CE32 console recordings).

• Spike sorting, DeepLabCut, YOLO, sharp-wave-ripple detection, place-cell metrics, theta sweeps, and neural manifold topology.

Project Fabrication

Long-term neural recordings across memory circuits with high-density flexible probes

In prep / unpublished manuscript

Polymer-based flexible thin-film probes to reduce tissue–device mismatch and support stable long-term single-neuron tracking across memory circuits.

Project page

• Large-scale hippocampal electrophysiology in behaving mice across environments; experimental design with Arduino, DLC, perfusions, and imaging.

• Population-dynamics analysis from high-dimensional data; single-cell place-cell stability across ~300-day recordings via spike sorting.
